- Feature column may have parent and subfeatures together. For example, USB is a parent feature and Samba is a subfeature of USB
- Parent feature and following subfeatures are divided by '/' character
- The first path existing in Paths column may be used to understand whether feature is installed to the device
- Look at exclude array listed in input.json to see which features are excluded from this release
